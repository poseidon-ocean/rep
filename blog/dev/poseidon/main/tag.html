HTML注释：

    可以将注释插入 HTML 代码中，这样可以提高其可读性， 使代码更易被人理解。浏览器会忽略注释，也不会显示它们。

    语法：%lt;!-- This is a comment -->
开始括号之后（左边的括号）需要紧跟一个叹号，结束括号之前（右边的括号）不需要

    合理地使用注释可以对未来的代码编辑工作产生帮助。

标题标签：

    可以将注释插入 HTML 代码中，这样可以提高其可读性， 使代码更易被人理解。浏览器会忽略注释，也不会显示它们。

    <h1>-<h6>标签定义标题

    其中<h1>定义最大标题，<h6>定义最小标题

<!-- h1 - h6标签字体大小越来越小 权重比例也越来越低 网页布局按规范使用h标签，一个页面只能使用一个h1标签，其他的可以使用多个 利于搜索引擎优化，使用多个h1百度会认为是作弊，非法的 -->

样例：h1-h6

段落标签：

    <p>标签定义段落

    可以将HTML文档分割成若干段落

加粗标签：

    <strong>和<b>都能加粗文本，但其中<strong>表示强调文本

    b标签只是用来加粗效果，strong标签不仅是加粗， 搜索引擎可以识别

    在 HTML 4.01 中，<strong> 定义语气更重的强调文本，但是在 HTML 5 中，<strong> 定义重要的文本。

斜体标签：

    <em>和<i>都能定义斜体字，但其中<em>起强调作用


提示和注释

    注释：根据 HTML5 规范，在没有其他合适标签更合适时，才应该把 <b> 标签作为最后的选项。 HTML5 规范声明：应该使用 <h1> - <h6> 来表示标题，使用 <em> 标签来表示强调的文本， 应该使用 <strong> 标签来表示重要文本，应该使用 <<mark> 标签来表示标注的/突出显示的文本。
    提示：您也能够使用 CSS "font-weight" 属性来设置粗体文本。

超链接：

<a>标签可以从通过点击从一个页面跳到另一个页面
语法：<a href=“url”>文字显示</a>

a标签的属性：href="要跳转的目标链接"
target="_blank" 是否新开一个浏览窗口，如果不添加默认当前页面打开
href里边的常用值：#（当前页面不跳转） javascript; javascript:void(0)
rel="nofollow"不允许搜索引擎追踪收录

baidu.com顶级域名 www.baidu.com一级域名 baike.baidu.com二级域名 m.baike.baidu.com三级域名

锚标记：

相当于书签，当使用锚标记时，我们可以创建直接跳至该命名锚的链接
语法：<a name=“自定义名字">锚（显示在页面上的文本）</a> 创建使用该锚的链接：<a href=“#自定义的锚名字”>显示内容</a>

图片标签：

<img>标签将其他文件夹或服务器的图片显示到网页中

图片标签img元素向网页中嵌入一幅图像，是从网页上连接图像

图片标签四要素：src路径，alt描述,width宽度，height高度
alt只是和SEO有关系，百度不能识别图像，通过alt来辨别并收录
如果不添加宽度高度，页面加载会计算图片的宽高，所以加入width和height ，而且只能是行内加入，不能当样式加入，不需要单位
title=""这个属性是提高客户体验的
大型网站公司都是这样的标准

语法：<img src=“图片地址” alt=“自定义替代文本">
<img>标签只包含属性，并且没有闭合标签

<frameset>标签：

定义一个框架集 

语法：
<frameset  cols=“25%,50%,25%>
	<frame src=“url1”/>
	<frame src=“url2”/>
	<frame src=“url3”/>
</frameset>

     frameset 元素可定义一个框架集。它被用来组织多个窗口（框架）。每个框架存有独立的文档。在其最简单 的应用中，frameset 元素仅仅会规定在框架集中存在多少列或多少行。您必须使用 cols 或 rows 属性 垂直框架(cols)、水平框架(rows)

混合框架结构：

<frameset rows="50%,50%">

	<frame src=" frame_a.html">

	<frameset cols="25%,75%">
	<frame src=" frame_b.html">
	<frame src=" frame_c.html">

</frameset>

导航框架

<frameset cols="120,*">
	<frame src="html_contents.html">
	<frame src="/frame_a.html" name="showframe">
</frameset>

内链框架：

<iframe src="/i/eg_landscape.jpg"></iframe>
